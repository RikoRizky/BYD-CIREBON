# BYD Cirebon Website

Website informasi resmi untuk showroom BYD di Cirebon. Aplikasi ini menampilkan berbagai model mobil BYD, daftar harga, fitur, dan informasi kontak.

## Fitur Utama

- **Halaman Utama (Hero)**: Tampilan menarik dengan carousel mobil BYD.
- **Tentang Kami**: Informasi tentang BYD dan showroom di Cirebon.
- **Fitur Mobil**: Penjelasan fitur-fitur unggulan mobil BYD.
- **Daftar Harga**: Harga terbaru untuk setiap model mobil.
- **Detail Unit**: Informasi detail untuk setiap model, termasuk gambar, spesifikasi, dan warna.
- **Kontak**: Formulir kontak dan informasi lokasi showroom.
- **Footer**: Link ke media sosial dan informasi tambahan.

## Teknologi yang Digunakan

- **Frontend**: React.js dengan Vite sebagai build tool.
- **Routing**: React Router DOM untuk navigasi antar halaman.
- **Styling**: Tailwind CSS untuk desain responsif.
- **Animasi**: GSAP untuk animasi halus.
- **Deployment**: Hostinger dan Vercel untuk hosting online.

## Struktur Project

```
byd-cirebon/
├── public/                 # Asset statis (gambar, PDF brochure)
├── src/
│   ├── components/         # Komponen React (Hero, About, dll.)
│   ├── data/               # Data statis (harga, carousel, dll.)
│   └── main.jsx            # Entry point React
├── dist/                   # Output build (dihasilkan otomatis)
├── vercel.json             # Konfigurasi Vercel untuk SPA
├── package.json            # Dependencies dan scripts
└── README.md               # Dokumentasi ini
```
